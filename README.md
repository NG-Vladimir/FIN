# Финансы — личный учёт

Мобильное веб-приложение для учёта доходов и расходов. Открываешь по ссылке — всё работает в браузере, данные хранятся только на устройстве (localStorage). Никаких серверов и баз данных.

## Как открыть

1. Залей папку на GitHub (или любой хостинг статики: GitHub Pages, Netlify, Vercel).
2. Включи GitHub Pages: **Settings → Pages → Source: main branch** (или выбери ветку и папку `/ (root)`).
3. Открой ссылку вида `https://твой-username.github.io/имя-репо/` — это и есть приложение.

Можно также открыть локально: просто открой файл `index.html` в браузере или подними локальный сервер:

```bash
# Python
python3 -m http.server 8000

# или npx
npx serve .
```

Затем открой в браузере `http://localhost:8000`.

## Возможности

- **Доходы и расходы** — кнопки «Доход» и «Расход», ввод суммы, категории и заметки.
- **Баланс** — автоматический подсчёт общего баланса, суммы доходов и расходов.
- **История** — список операций с возможностью удаления.
- **Только локально** — все данные в `localStorage` браузера, никуда не отправляются.

## Технологии

- HTML5, CSS3 (адаптивная вёрстка, тёмная тема по системным настройкам), JavaScript (без фреймворков).
- Подходит для добавления на главный экран телефона (PWA-подобное поведение через meta-теги).
